Структура головної сторінки WEB-застосунка, яка містить HTML-код таблиці, зображення, списків, посилань


HTML-код таблиці та скріншот цієї таблиці. Додати теорію про теги та атрибути, які застосовуються при створенні таблиць



У цій таблиці використовуються наступні теги та атрибути:

  • table - визначає таблицю в HTML.
  • thead - контейнер для заголовків стовпців таблиці, який містить tr із заголовками (th).
  • tbody - контейнер для основного вмісту таблиці, який містить рядки таблиці (tr), що складаються з комірок (td).
  • tr - представляє рядок у таблиці.
  • th - визначає заголовок стовпця в таблиці.
  • td - визначає комірку в таблиці для основного вмісту.
